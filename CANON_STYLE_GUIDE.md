# SPACEFROGS CANON STYLE GUIDE

## Visual Identity

SPACEFROGS should look like premium dark sci-fi.

Core look:
- emerald green visor glow
- cyan HUD accents
- black / graphite / wet dark surfaces
- CRT scanlines
- tactical UI panels
- living biomech armor
- frog-first anatomy
- cinematic fog / swamp / neon / signal atmosphere

## Hero Design Logic

SpaceFrogs are athletic amphibious biomech operators.

They should not look like:
- Halo soldiers
- Iron Man suits
- comedy frogs
- generic aliens
- toy-like mascots

They should look like:
- frog-headed tactical heroes
- living armor operators
- biological + technological
- premium collectible action figures
- game-ready characters

## Class DNA

Leader / Command:
Balanced hero presence, command-core, controlled authority.

Ranger / Recon:
Slim, fast, agile, scanning predator.

Heavy / Support:
Compact dense support platform, stronger shoulders, arms and chest, lower center of gravity.

Tech / Specialist:
Modular signal/interface body, utility nodes, tools and diagnostics.

Sniper / Precision:
Narrow, calm, surgical silhouette, optics-led design.

Assault / Breach:
Forward-pressure body, kinetic, brutal but mobile.

## UI Style

The app should feel like:
- tactical command console
- Kickstarter launch deck
- fake SpaceFrogs OS
- mission control panel
- cinematic game menu

Use:
- glass cards
- glowing borders
- animated scanlines
- mission labels
- signal language
- subtle HUD motion
- strong CTA buttons

## Colors

Primary:
Emerald green

Secondary:
Cyan

Background:
Black / graphite / deep swamp dark

Accent:
Small amber/red only for warnings or enemy threat.

## Typography Feel

Use strong sci-fi headings.
Use clean readable body text.
Avoid overly decorative fantasy fonts.

## Forbidden Direction

Do not make SPACEFROGS:
- childish
- goofy
- generic superhero
- generic alien
- military-only
- over-mech
- too human
- too cartoonish

SPACEFROGS should always feel like a premium tactical mythic sci-fi universe.
