# SPACEFROGS Kickstarter PWA

SPACEFROGS is a premium sci-fi tactical franchise about amphibian warriors in living biomech armor fighting the Θ9 machine threat.

This repo contains the mobile-first Kickstarter landing app for:

## SPACEFROGS: First Contact Pack

The app should feel like:
- cinematic tactical command interface
- dark emerald/cyan CRT HUD
- premium collectible franchise launch
- Kickstarter preview page
- installable PWA

## Core Product

First Contact Pack may include:
- Quickstart rules
- Mission 0 / First Contact
- Hero squad cards
- Enemy cards
- Tokens
- Tactical map
- Lore booklet / Codex
- Digital extras

## Brand Rules

SPACEFROGS is not goofy.

Tone:
- serious
- tactical
- mythic
- cinematic
- premium
- emotionally powerful

Visual style:
- dark sci-fi
- emerald visor glow
- cyan HUD lines
- CRT scanlines
- frog-tech
- biomech armor
- Θ9 machine threat
