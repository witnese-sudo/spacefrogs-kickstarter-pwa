# SPACEFROGS™ MISSION 0 — BOARD LAYOUT + SETUP v1.0
## First Contact / Relay Breach at the Swamp Edge

STATUS: Playtest draft
MODE: Co-op PvE
PLAYERS: 1–4
BOARD SIZE: 12×12 starter grid / zones

---

## 1. Board Concept

Mission 0 uses a compact battlefield so new players can learn fast.

The board should feel like:

- Swamp edge entry zone
- Broken relay in the center
- Machine patrol pressure from the far side
- A few cover points
- A clear extraction route
- Enough danger to force choices, not enough rules to slow the first test

---

## 2. Grid Coordinates

Use a 12×12 grid.

Columns:
A B C D E F G H I J K L

Rows:
1 at Frog entry edge
12 at Machine edge

Frogs start near Row 1.
MachineUnits09 begin near Row 10–12.
Relay Objective sits near center.

## 4. Deployment

### Frogs
Place 3 Frogs anywhere in Frog deployment zones:
- C1–D2
- I1–J2

Recommended:
- Leader at D2
- Ranger at C1 or I1
- Tech Specialist at J2 or D1

### Machines
Place 3 Watcher Drones:
- D12
- E10
- H10 or I12

Optional:
- Pursuit Hound enters at Alert 5 from one of these edges:
  - A12
  - L12
  - F12 / G12 gate

---

## 5. Relay Objective

Place Relay Objective across these central cells:

```text
F6, G6, F7, G7
```

A Frog is adjacent to the Relay if it is touching any of these cells orthogonally or diagonally.

Relay needs:
- 3 Decode Progress

---

## 6. Cover Rules

Cover cells represent swamp rocks, broken roots, crashed machine parts or ruined tech.

Simple first-test rule:

If a Frog is adjacent to or inside Cover:
- It may gain +1 defense reroll against one attack.

If this is too strong:
- Cover only works if the attack line crosses the cover cell.

If this is too complex:
- Just say “standing in cover gives +1 defense reroll.”

---

## 7. Noise Token Placement

Place Noise tokens on the cell where the noisy action happened.

Noise is created by:
- Dash
- Ranger Leap Dash
- Non-Tech decode
- Optional loud weapon / dramatic action

Watcher Drones move toward nearest Noise token if they cannot shoot a Frog.

After a Drone reaches a Noise token:
- Remove that Noise token.

---

## 8. Extraction

After the Relay reaches 3 Decode Progress, place Extraction Marker on either:

Easy mode:
- Any Frog deployment zone

Standard mode:
- C1–D2 only

Hard mode:
- Opposite side of starting deployment, I1–J2 if Frogs started on C/D side, or C1–D2 if Frogs started on I/J side

Victory requires:
- Relay decoded
- At least 2 Frogs extract

---

## 9. Difficulty Tuning

### Easy First Test
- Use 2 Watcher Drones
- Alert 6 only causes final warning
- No Pursuit Hound

### Standard Test
- Use 3 Watcher Drones
- Alert 5 summons optional Pursuit Hound
- Alert 6 causes lockdown

### Hard Test
- Use 3 Watcher Drones + Pursuit Hound at Alert 4
- Drones get aggressive at Alert 2
- Relay requires 4 Decode Progress
