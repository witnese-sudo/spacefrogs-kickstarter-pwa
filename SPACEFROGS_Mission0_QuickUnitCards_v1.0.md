# SPACEFROGS™ MISSION 0 — QUICK UNIT CARDS v1.0
## For Quickstart Mission 0 — First Contact

These are lightweight first-play cards. They are not final balance.

Use these to make Mission 0 playable without needing a full rulebook.

---

# FROG UNIT: LEADER
## Core role: Command / anchor / morale

MOVE: 5
HEALTH: 8
DEFENSE: 4+
RANGE: Medium
ATTACK: 2 dice

### Ability — Command Pulse
Once per round, choose one Frog within 4 spaces / zones.
That Frog may reroll 1 failed die.

### Ability — Hold the Line
If a Frog within 4 spaces / zones would be downed, the Leader may spend 1 action to give that Frog +1 defense reroll.

### Playtest Note
The Leader should feel like the squad’s stabilizer.
Not the strongest damage unit — the one that keeps the signal together.

---

# FROG UNIT: RANGER
## Core role: Movement / flank / pressure

MOVE: 7
HEALTH: 6
DEFENSE: 5+
RANGE: Short–Medium
ATTACK: 2 dice, or 3 dice at close range

### Ability — Leap Dash
Once per round, Ranger may move +2 extra spaces / zones.
If used, add 1 Noise token.

### Ability — Close Strike
When attacking at close range, roll +1 attack die.

### Playtest Note
The Ranger should feel fast, agile and cinematic.
If the Ranger feels too weak, increase close attack pressure.
If too strong, make Leap Dash always add Noise.

# FROG UNIT: TECH SPECIALIST
## Core role: Objective / relay decode / support

MOVE: 5
HEALTH: 6
DEFENSE: 5+
RANGE: Medium
ATTACK: 1 die

### Ability — Clean Decode
When adjacent to the Relay Objective, spend 1 action to add 1 Decode Progress without adding Noise.

### Ability — Signal Patch
Once per round, remove 1 Noise token within 4 spaces / zones.

### Ability — Emergency Spark
Once per mission, the Tech may cancel 1 Alert increase.

### Playtest Note
The Tech must feel important even with weak combat.
If players ignore the Tech, make the Relay harder for non-Tech Frogs.

# MACHINE UNIT: WATCHER DRONE
## Core role: Patrol / scan / pressure

MOVE: 5
HEALTH: 3
DEFENSE: 5+
RANGE: Medium
ATTACK: 1 die

### AI Priority
On Machine turn, each Watcher Drone follows this order:

1. If a Frog is visible and in range: shoot.
2. If a Noise token exists: move toward nearest Noise token.
3. If no Noise token exists: move toward the Relay Objective.
4. If adjacent to Relay: scan it and add +1 Alert.

### Alert 3 Upgrade
At Alert 3+, Watcher Drones become aggressive:
If they can see a Frog, they shoot before moving.

### Playtest Note
Watcher Drones should feel readable but relentless.
They are not smart like people. They are protocol hunters.

# OPTIONAL MACHINE UNIT: PURSUIT HOUND
## Core role: Fast fear / endgame pressure

MOVE: 8
HEALTH: 5
DEFENSE: 4+
RANGE: Melee / Close
ATTACK: 3 dice

### Enter the Mission
Only use the Pursuit Hound if:
- Players learn fast
- Alert reaches 5
- You want stronger cinematic pressure

### AI Priority
1. Move toward nearest visible Frog.
2. If no Frog visible, move toward nearest Noise token.
3. If adjacent to a Frog, attack.
4. If a Frog is downed, move toward the next closest Frog instead of staying still.

### Ability — Predator Leap
Once per Machine turn, the Hound may ignore one terrain penalty while moving.

### Playtest Note
The Hound should feel scary.
If it wipes the squad too fast, reduce attack to 2 dice.
If it feels too weak, let it enter earlier at Alert 4.

# QUICK REFERENCE

## Frog Turn
Each Frog gets 2 actions:
- Move
- Shoot
- Decode Relay
- Interact
- Take Cover
- Revive adjacent downed Frog
- Dash

## Machine Turn
Activate all Machine units using their AI priority.

## Alert Increases When
- Frog dashes
- Non-Tech Frog decodes
- Drone scans Relay
- Frog is downed
- Optional: players take too long

## Relay Objective
Needs 3 Decode Progress.

## Victory
Decode relay + extract at least 2 Frogs.

## Defeat
2+ Frogs downed, Alert reaches 6 before decode, or Frogs fail to extract.
