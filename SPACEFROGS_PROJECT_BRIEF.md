# SPACEFROGS PROJECT BRIEF

## Project

SPACEFROGS is an original premium sci-fi tactical franchise by Klaus Wittendorff.

It combines:
- tactical tabletop missions
- collectible hero figures
- cinematic lore
- digital game potential
- merchandise
- Kickstarter-first launch strategy

## Core Concept

SPACEFROGS are amphibian warriors in living biomech armor.

They are not goofy cartoon frogs.
They are serious, tactical, mythic, premium sci-fi heroes.

## Main Enemy

The main machine enemy is Θ9 / Theta 9.

Θ9 represents:
- cold machine intelligence
- signal corruption
- assimilation
- tactical enemy pressure
- machine swarm threat
