# SPACEFROGS™ SECTOR 7F FPS CANON RULES v1.0

Locked source of truth for the SpaceFrogs First Signal first-person shooter direction.

## Core Law

**Frogs first. Biotech second. Machine third.**

SpaceFrogs must never become goofy, cute, generic sci-fi, or a human body with a frog head. They are serious, tactical, mythic amphibian operators in living biotech armor.

## Sector 7F Identity

Sector 7F is a dark, wet, biotech-industrial combat zone.

Look:

- rain-wet black metal
- dark concrete
- emerald/cyan/purple bioelectric veins
- alien biotech cables
- reactor fog
- storm light
- survival shooter mood
- brutal industrial wetness

It must feel like a real playable game level, not a picture, poster, flat AI scene, or fake environment render.

## FPS Arena Law

Sector 7F FPS must be built as a playable arena:

- walkable floors
- central reactor pit/objective
- upper catwalk loops
- ramps and stairs
- side corridors
- tunnel gates
- cover blocks
- vertical combat layers
- clear sightlines
- flank routes
- sniper/overwatch positions
- enemy spawn doors
- objective pressure

Map flow can be inspired by classic tactical shooter map logic, but it must not feel like a copy. It is SpaceFrogs Bioforge warfare.

## Unreal / GLB Production Law

All 3D assets must be designed as game-ready modular pieces.

Arena prompts must always include:

- Unreal Engine ready
- modular geometry
- FPS scale
- clean collision-friendly shapes
- reusable wall/floor/catwalk/cover modules
- no text
- no UI
- no poster frame
- no flat image plane
- no characters in arena asset
- no weapons unless specifically making weapons or FPS arms later

## Sector 7F Gameplay Loop

1. Player chooses Frog operator.
2. Player enters Bioforge Breach.
3. Θ9 waves activate.
4. Player reaches central reactor.
5. Player holds/scans/charges reactor.
6. Waves escalate.
7. Player clears final pressure or extracts.

The game feeling:

- tactical pressure
- wet survival
- reactor objective tension
- frogs holding the line against dead machine logic

## First Five FPS Operators

### WIT — Commander

Balanced, iconic starter, command presence, emerald signal identity.

### SKIP — Ranger

Fast, agile, recon/flank movement, cyan scanner energy.

### BULWARK — Heavy

Dense compact support frog, highest health, holds reactor under pressure.

### PATCH — Engineer

Objective specialist, fastest scan/repair/reactor interaction, violet/cyan utility nodes.

### VEIL — Sniper

Narrow, calm, surgical, catwalk overwatch, precision identity.

## Θ9 Enemy Canon

Θ9 enemies are not random robots. They are cold machine corruption trying to replace living signal.

Core FPS enemies:

- Θ9 Drone: group pressure, scanner eye, patrol/chase behavior
- Θ9 Hound: fast low predator, rushes player, close threat
- Θ9 Construct: future heavy lane blocker
- Θ9 Nodes: corruption/objective pressure
- Θ9 Wave Director: battlefield intelligence, not spawn spam

## Visual Forbidden List

Never make Sector 7F:

- cute
- comedy frog
- toy mascot
- fantasy goblin swamp
- generic Halo
- generic cyberpunk
- generic military bunker
- Iron Man frog armor
- clean shiny showroom sci-fi
- flat AI concept poster pretending to be a level

## Prompt Rule

When generating Sector 7F assets, always include:

```text
Build it as a real playable 3D game environment with FPS scale, modular walkable geometry, central reactor objective, cover, ramps, catwalks, side corridors, tunnel gates, vertical combat layers, wet biotech-industrial materials, glowing emerald/cyan/purple bioelectric veins, and clean Unreal Engine-ready GLB structure.
```

## Production Rule

Do not jump to a giant game first.

Order:

1. playable browser prototype
2. Sector 7F FPS canon docs
3. Tripo/GLB modular arena assets
4. Unreal First Person project
5. greybox arena
6. reactor objective
7. WIT playable
8. Θ9 drone + hound
9. wave director
10. visual polish
11. trailer/demo build

One arena first. Make it alive. Then expand.
