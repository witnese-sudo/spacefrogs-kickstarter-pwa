```text
Style: SpaceFrogs Sector 7F, rain-wet black metal, dark industrial concrete, glowing green/cyan/purple bioelectric veins, alien biotech cables, reactor energy, gritty survival shooter mood, stormy blue-green lighting.

Game production requirements: clean modular geometry, FPS scale, playable paths, clear combat lanes, cover positions, elevated sniper/catwalk areas, open central objective zone, reusable wall/floor/catwalk/cover modules. Make it suitable as a GLB environment blockout with HD textures.

No characters, no weapons, no text labels, no UI, no poster frame, no flat image plane, no decorative-only unusable geometry.
```
