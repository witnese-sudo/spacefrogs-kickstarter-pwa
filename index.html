Sector 7F // Wet Bioforge Warzone

The battlefield is alive.

Sector 7F is a rain-wet industrial Bioforge where walls breathe with green signal veins, floor plates sweat with corrupted coolant and every objective can wake the machine. This is not generic space war. This is frogs-first survival tactics inside a living system that wants to overwrite life.

Frogs first

What is SpaceFrogs?

A frog-first sci-fi survival universe built around awakened amphibian operators, living signal and Θ9 machine corruption.

Board game or video game?

It begins as a tactical co-op tabletop miniatures game, with digital tools and Unreal/3D exploration as future support layers.

How many players?

The core playtest direction is 1–4 co-op PvE, with Mission 0 tuned for fast intro testing.

Five premium frog heroes, five enemy wave types, missions, cards, tokens, quickstart rules and modular Sector 7F board tiles.
