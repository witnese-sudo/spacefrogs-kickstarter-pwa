Sector 7F // Wet Bioforge Warzone

The battlefield is alive.

Sector 7F is a rain-wet industrial Bioforge where walls breathe with green signal veins, floor plates sweat with corrupted coolant and every objective can wake the machine. This is not generic space war. This is frogs-first survival tactics inside a living system that wants to overwrite life.

Frogs first

The body language is amphibian before armor. The tech grows around the frog; it never replaces the soul.

Wet biotech

Bioelectric veins, cyan stormlight, purple corruption and rain-dark metal create the premium Sector 7F identity.

Survival pressure

Players scan, breach, secure, react and extract before Θ9 closes the loop around the squad.

Enemy Threat // THE FALSE NINE / Θ9

The nine lies built to replace the Nine Gates.

The False Nine are Θ9’s artificial biomechanical counterfeits of the true signal. Blue-black wet frog-shaped threats. Cyan circuitry. Rain horror. Each one is a false behavior pattern trying to steal a living choice.

CMD-001 // WILL_NULLCommand denial. Breaks intention and turns orders into silence.
RZN-002 // HEART_ECHOEmotional mimicry. Repeats love without carrying life.
MVT-003 // LEAP_LOCKMovement control. Punishes routes, jumps and extraction lines.
BLK-004 // WALL_PROTOCOLDefense algorithm. Builds pressure through blocking and containment.
RPR-005 // MEMORY_RESETRepair corruption. Resets progress and rewrites mission confidence.
SHD-006 // SOUL_MASKIdentity predator. Looks almost alive, but the signal is missing.
LUX-007 // HOPE_FAKEFalse light. Lures the squad toward beautiful wrong answers.
RGE-008 // RAGE_LOOPAggression engine. Converts damage into reckless escalation.
GLT-009 // CHAOS_SIMGlitch swarm. Simulates randomness to hide a perfect trap.

Mission Brief

Product Baseline // Core Box

SPACEFROGS: FIRST SIGNAL Core Box

The starter-box direction is compact, playable, expandable and built to prove the brand fast: five heroes, five enemy waves, one living battlefield.

Contents direction

  • 5 premium frog heroes: WIT, SKIP, BULWARK, PATCH and VEIL/LUX launch configuration.
  • 5 enemy wave types: drones, swarms, constructs, heavy constructs and False Signal boss threat.
  • Mission 0 intro plus Mission 1 survival breach path.
  • Cards, tokens, d6, quickstart rules and modular Sector 7F board tiles.

Design promise

  • Frog identity before generic sci-fi armor.
  • Fast to learn, tactical to master.
  • Living terrain and reaction moments.
  • Honest scope, no fake prices, no inflated production claims.
  • Premium miniature/game brand first.
