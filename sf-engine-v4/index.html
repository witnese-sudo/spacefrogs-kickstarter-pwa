What is SpaceFrogs?

Frogs first. Biotech second. Machine third.

SPACEFROGS is a serious tactical sci-fi universe about awakened amphibian warriors in living biomech armor fighting the Θ9 machine threat. It is built for tabletop missions, collectible heroes, cinematic lore, digital tools and a future Kickstarter launch.
